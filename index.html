Page d'acceuil

Bienvenue sur le site web regroupant un ensemble de traitement de BATIROC BPL.

Ce site web permet de récupérer des données externes à Batiroc, notamment via des API, mais également à réaliser des traitements via du dépôt de documents par exemple.

Vous pourrez trouver l’ensemble des traitements à gauche. Si vous souhaiter consulter la documentation, vous pouvez cliquer sur le bouton « documentation », associé à chaque traitement.
